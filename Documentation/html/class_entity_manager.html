std::unordered_map< int, Entity > entities
 Stores all entity objects by their unique identifier (entityUID). entities is an unordered_map that stores every Entity created by the EntityManager. Storing each Entity by it's entityUID allows for fast Entity lookup by other Systems in the engine.
 

Detailed Description

System class that orchestrates all Entity handling. The EntityManager system manages Entity creation, composition, and destruction. Each entity created by the EntityManager is stored in an unordered map by it's assigned unique identifier (entityUID).
