\doxysubsection{Detailed Description}
\doxylink{class_animation}{Animation} component. 

The \doxylink{class_animation}{Animation} component tells the \doxylink{class_animation_engine}{Animation\+Engine} that an entity contains animation data. It stores the frames associated with animation cycles, and it stores each animation cycle itself.
