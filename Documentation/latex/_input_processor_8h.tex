\doxysubsection{Detailed Description}
Header file for the \doxylink{class_input_processor}{Input\+Processor} class in Bloom Engine. 

The \doxylink{class_input_processor}{Input\+Processor} class is responsible for handling user input and dispatching appropriate events to the rest of the engine. It interprets SDL events and translates them into engine-\/specific events that are then dispatched through the \doxylink{class_dispatcher}{Dispatcher}.
