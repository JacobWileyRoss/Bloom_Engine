\doxysubsection{Detailed Description}
Central class of the Bloom Engine. 

The \doxylink{class_core}{Core} class is responsible for orchestrating the engine\textquotesingle{}s subsystems, managing the main loop, and handling initialization and shutdown processes. It provides the primary interface for running the game engine.
