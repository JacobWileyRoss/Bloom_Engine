\doxysubsection{Detailed Description}
Processes user input and dispatches engine events. 

The \doxylink{class_input_processor}{Input\+Processor} is a key component of the Bloom Engine that listens for SDL input events, interprets them, and then creates and dispatches corresponding engine events to influence the game state or behavior of entities.
